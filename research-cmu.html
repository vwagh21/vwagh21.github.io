Research

LLM-Guided Adaptive Control for Exoskeletons based on real-time user feedback and scene context

LLMs on edge: Using small LLMs with supervised finetuning, few shot and prompt engineering to map qualitative user feedback to exoskeleton control parameters.

Multimodal Pipeline: Using Whisper tiny for audio transcription and depth + segmentation information from stereo cameras for extracting scene context to guide LLM response by scene context enriched prompting.

Latency: Optimizing models using quantization, scripts through multiprocessing and lightweight models for real-time personalization.

Features: Including user-specific memory, continual learning pipelines, safety guardrails and ambiguity handling frameworks.

Personalized Torque Prediction for Stroke Populations using minimal user data

Dataset generation: Used musculoskeletal simulation software (OpenSim) with its MATLAB API to automate and generate simulated IMU readings from Motion Capture datafiles of two heterogenous datasets (35 subjects) with separate IMU configurations, bringing them into the same configuration.

Model training: Built and trained a Temporal Convolutional Neural Network on large-scale, healthy-subject locomotion data, and finetuned it with data from post-stroke subjects for personalization with 0.2% subject-specific data.

Publication: Presented a poster at American Society of Biomechanics 2025 showing effective adaptation from able-bodied data across multiple tasks using 20 seconds of single-task data per subject.

Evaluating MediaPipe for Post-Stroke Kinematic Assessment

Upper limb reaching task: Analyzed upper-limb kinematics during a gamefied reaching task captured for post-stroke subjects.

Methodology: Used OpenCV and MediaPipe to extract body poses and analyze kinematics from ~0.14M actions & ~0.3M frames, calculated statistical correlations between observed kinematics and maladaptive compensation strategies.

Publication: Published a paper in Journal of Neuroengineering and Rehabilitation supporting the use of off-the-shelf toolkits like MediaPipe for clinical kinematic assessment of post-stroke recovery.

Evaluating MediaPipe for Fine Upper-Limb Motion Capture

Study: Used a 2D tracing task to validate low-cost hand tracking vs. traditional motion capture.

Outcome: Analyzed 16K fine motion trajectories of extremely fast, dextrous fine upper limb movement (avg 2 second duration), ~70% accuracy between low cost tool and standard; supporting scalable clinical monitoring use.

Publication: Published a paper in Journal of Medical Internet Research: Formative research.
